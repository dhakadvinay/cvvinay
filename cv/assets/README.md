# Assets Directory

This directory contains all images and static files for the portfolio website.

## Required Images

Please add the following images to this directory:

### Profile Images
- `profile.jpg` - Hero section profile picture (recommended: 400x400px, square)
- `about.jpg` - About section image (recommended: 600x400px, landscape)

### Portfolio Project Images
- `project1.jpg` - Portfolio project 1 (recommended: 800x450px, 16:9 ratio)
- `project2.jpg` - Portfolio project 2
- `project3.jpg` - Portfolio project 3
- `project4.jpg` - Portfolio project 4
- `project5.jpg` - Portfolio project 5
- `project6.jpg` - Portfolio project 6

### Favicon & Social
- `favicon.ico` - Website favicon (16x16px or 32x32px)
- `og-image.jpg` - Open Graph image for social sharing (1200x630px)

## Image Optimization Tips

1. **Format**: Use WebP format for better compression, or JPG for compatibility
2. **Size**: Optimize images before uploading (use tools like TinyPNG, ImageOptim)
3. **Naming**: Use lowercase with hyphens (e.g., `project-1.jpg`)
4. **Alt Text**: Remember to update alt text in `index.html` for accessibility
